# Portafolio Web | Pedro Silva Rojas

Este es el repositorio oficial de mi portafolio profesional, desarrollado bajo el nombre de marca **NexoWeb**. El sitio ha sido diseñado con una estética **Neo-Brutalista** y está optimizado para ofrecer una experiencia de usuario fluida, accesible y de alto impacto visual.

## 🎯 Propósito

El objetivo de este proyecto es centralizar mi trayectoria como **Product Builder**, integrando mis facetas de Desarrollador Frontend, Product Owner y especialista en Customer Experience (CX). El portafolio no solo muestra mis proyectos técnicos, sino que también comunica mi propuesta de valor: conectar objetivos de negocio con tecnología funcional y accesible.

## 🚀 Características Principales

- **Diseño Responsivo:** Adaptado completamente para dispositivos móviles, tablets y escritorio con menús de navegación específicos.
- **Estética Moderna:** Estilo Neo-Brutalista con tipografías potentes, colores vibrantes sobre modo oscuro y animaciones sutiles.
- **Accesibilidad (A11y):** Implementación de estándares WCAG, incluyendo navegación por teclado, etiquetas ARIA y contrastes adecuados.
- **Navegación Intuitiva:** Sistema de navegación dual (Lateral y Superior) para facilitar el acceso a Proyectos, Sobre Mí y Experiencia.
- **Optimización de Carga:** Uso de formatos de imagen modernos (WebP) y código limpio sin dependencias pesadas.
- **Integración Social:** Accesos directos a WhatsApp, LinkedIn y GitHub.

## 🛠️ Requisitos Previos

Al ser un proyecto de tecnologías web estándar, no requiere de procesadores ni compiladores especiales. Solo necesitas:
- Un navegador web moderno (Chrome, Firefox, Edge, Safari).
- (Opcional) Una extensión de servidor local para VS Code como **Live Server**.

## 💻 Instalación y Uso

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/Pedro-Silva-R/portfolio.git
   ```

2. **Acceder a la carpeta:**
   ```bash
   cd portfolio
   ```

3. **Ejecutar el proyecto:**
   Simplemente abre el archivo `index.html` en tu navegador o haz clic derecho en el archivo dentro de tu editor y selecciona "Open with Live Server".
